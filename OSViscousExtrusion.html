A LOW COST OPENSOURCE COMPLETE WORKFLOW FOR VISCOUS MATERIAL EXTRUSION

This project started as conclusion obtained during the development of Wood 3D Printer.

With the goal of massifying the use of biocomposites in Digital Fabrication laboratories I seek to develop a whole workflow for viscous material extrusion. This project intends to develop a simple, modular system that can be installed in any commercial 3D printer or 3 axis CNC machine.

Even though there's plenty of work done in the development of Open Hardware extrusion systems for viscous materials as clay. None of them consider a whole system; Slicer, extrusion system (hardware and firmware) and material recipes. Its important to coinsider theres an interdepencie in this 3 variables, and any modification in one of them will afect the others:

This 3 key valuables show a dependency between each other, and cannot be modified without affecting the others. Currently there is not a firmware for extrusion system control that enables the easy control and communication of these 3 parameters.

Open Source extrusion system commonly in work the following ways:

  • Through a mechanical feeder, through a stepper motor, controlled as a 4 axis CNC machine.

  • Through a pneumatic feeder, through an air pump user controlled independent from the machine control system.

  • Through any of the two previously mentioned feeders (user controlled), and an auger screw controlled as a normal 3D printer.

There isn’t a firmware, neither a protocol that enables the communication between the 3 axis machine and the extrusion system. In order to fix that I propose an architecture that would enable I2C communication from a Ramp1.4 , running Marlin4ViscousMaterial to a Biofeeder Arduino Shield.This architecture will allow the user to control the extrusion system variables in the same GCode. This will enable the development of one simple slicer specialized for Viscous Materials 3D printers.

This architecture allows the user to choose the extruder configuration he prefears, direct extrusion, or an extrusion system based on an auger screw. And the feeding system, mechanical or pneumatic.

This is kind of one of those never ending projects. It started in 2019 during my Fab Academy course, with the objective to develop an extrusion system for viscous biomaterials. The project continued with Arauco in Chile, always with the focus on the materials formulas and properties. Currently the project is froozen, I will go back to work on it in 2021 when Chilean universities re-open.
